# LUCA IMMERSIVE × Cravin' Adventure Studios — Case Study

A single-page case study of a live, interactive LUCA IMMERSIVE experience: the audience's own laser pointers and flashlights (on LUCA IMMERSIVE wristbands) fed through a GoPro into Resolume, where real-time effects were projection mapped onto the building with no delay, then emergent crowd play took over.

**Live:** https://cravinadventure.github.io/client-case-study-luca-immersive/

Self-contained static `index.html` (inline CSS + inline SVG). No build step.
